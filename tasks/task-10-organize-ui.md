# Organize UI

Now thw html page has one tool box, in top right corner. It contaons tools, like auto-foucus, follow position, clear; and settings - style and api key. They're logicaly different, we need to split settings and tools.

# Requirements

Tools must stay on screen, settings can go into another place, that opens by click on the button somewhere. Settings will include a list of predefined styles, a button to add a custom style, a button to change api key. Tools are just 3 buttons, like now.

As you can see, there is not many items, but they must be cleverly positioned on the screen: be in one place, but also logically separated. Also, the logo and the version must be somehere on the screen too.Overal, design must be minimalistic and functional.
